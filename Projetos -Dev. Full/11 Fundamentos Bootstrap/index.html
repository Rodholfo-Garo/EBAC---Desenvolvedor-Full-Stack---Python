O que é o Bootstrap?

Booststrap é um framework (abstração que une códigos) muito usado no desenvolvimento front-end, que fornece estruturas de CSS ja feitas, para a criação de sites e aplicações responsivas de forma rapida e simples.

Foi Desenvolvido para o Twitter por um grupo de desenvolvedores liderados por Mark Otto e Jacab Thornton e muito rápido se tornou uma das estruturas de front-end e projetos abertos mais populares do mundo.

Ele é encontrado no site oficial e npm, pode ser baixado usando o código fonte dele, ou por meio da própria rede de distribuição de conteúdo, conhecida como CDN.

Para instala-lo, acesse o site https://getbootstrap.com/, vá em Read the Docs, no menu lateral direito, selecione CDN links, encontraremos os links para o CSS e JS. Para para o uso do CSS, copie o link e cole no head do documento, agora, se o seu projeto for mais robusto e precisar de um MENU Suspenso, NAVbar, Alertas e mais algumas animaçãoe para o JS, copie o link e cole no final da pagina, antes da tag de fechamento do body.

Construção de layout com Boostrap

O Bootstrap trabalha com sistema de grids, onde construimos o layout a partir de linhas e colunas.Para construir um layout com Bootstrap,ultilizamos as classes container, row e col, seguindo a estrutura.

  • div class="container"
  • div class="row"
  • div class="col"
  • div class="col"
  • div class="col"
